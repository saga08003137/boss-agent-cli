合规边界

默认启用低风险辅助模式。打招呼（greet / batch-greet）、投递、联系方式交换、招聘者候选人搜索 / 简历 / 聊天、消息回复等敏感能力默认阻断并返回 COMPLIANCE_BLOCKED；需要时回到 BOSS 直聘平台官网由用户手动完成。

本地辅助 只读优先 用户主动触发 不规避风控 不批量触达 不抓取数据
核心能力

为 Agent 设计的每一处细节

welfare

福利精准筛选

--welfare 双休,五险一金 自动翻页补抓详情，按 AND 逻辑做真实匹配，而不是关键词命中。

schema · json

Schema 驱动 + JSON 信封

stdout 只输出 {ok, data, pagination, error, hints} 信封；boss schema 是能力真源。

shortlist

本地候选池

查看详情后本地保存、复盘候选岗位与漏斗统计；投递与沟通回到官网手动完成。

ai

AI 求职增强

JD 分析、简历润色、定向优化、模拟面试、沟通指导，兼容 OpenAI / Anthropic 接口。

platforms

多平台抽象

Platform / RecruiterPlatform 双注册表，--platform zhipin | zhilian | qiancheng。

agent-ready

四种 Agent 接入

Skill · subprocess · MCP · Python SDK，同一套能力，四条接入路径。

快速开始

跑通低风险闭环

# # 安装（浏览器内核仅用于登录 / 导出）
$ uv tool install boss-agent-cli
$ patchright install chromium

# # 跑通闭环
$ boss doctor
$ boss login
$ boss search "Golang" --city 广州 --welfare "双休,五险一金"
→ ok  15 个匹配（福利已 AND 命中）
$ boss shortlist add <security_id> <job_id>
$ boss stats
{
  "ok": true,
  "command": "search",
  "data": [ {
    "title": "高级 Golang 工程师",
    "welfare": ["双休", "五险一金"]
  } … ],
  "pagination": { "total": 15 },
  "hints": { "next_actions": ["boss detail …"] }
}
Agent 接入

同一套能力，四条路径

能力真源是 boss schema，支持导出 openai-tools / anthropic-tools 工具定义。

1 Skill 安装（推荐）

npx skills add can4hou6joeng4/boss-skill

2 subprocess + JSON

boss schema   # # 先读能力，再解析 stdout

3 MCP (Claude Desktop / Cursor)

uvx --from boss-agent-cli[mcp] boss-mcp

4 Python 直接嵌入

from boss_agent_cli import BossClient
