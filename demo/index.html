核心特性

不只是 API 封装，是完整的 Agent 求职基础设施

🔍

福利精准筛选

--welfare "双休,五险一金" 自动翻页逐个查详情，只返回真正匹配的结果。核心差异化功能。

🤖

Agent 友好协议

所有输出为 JSON 信封。boss schema 一次调用理解全部能力，hints.next_actions 引导下一步。

🧠

AI 简历优化

JD 匹配分析、简历润色、针对性优化、改进建议。支持 OpenAI 兼容的任意大模型。

🛡

智能反检测

基于 patchright 反检测浏览器 + 高斯分布请求延迟 + CDP 真实指纹，从根本上规避风控。

📊

求职流水线

pipeline 聚合沟通和面试数据，follow-up 筛选待跟进，digest 生成每日摘要。

📝

本地简历管理

多版本简历管理、HTML/PDF 导出、格式兼容导入、简历-职位关联追踪。

命令矩阵

覆盖求职全流程的 30+ 命令

boss schema工具能力自描述（Agent 首先调用）
boss login三级降级登录（Cookie / CDP / 扫码）
boss doctor环境诊断（依赖、网络、登录态）
boss search搜索职位（支持福利筛选 + 预设）
boss detail职位详情（httpx 快速通道）
boss recommend个性化推荐
boss greet向招聘者打招呼
boss batch-greet批量打招呼（上限 10）
boss apply发起投递/立即沟通（幂等）
boss chat沟通列表（支持多格式导出）
boss chatmsg聊天消息历史
boss chat-summary沟通结构化摘要
boss pipeline求职进度流水线
boss follow-up跟进提醒（超时/未读）
boss digest每日求职摘要
boss watch搜索订阅与增量监控
boss shortlist职位候选池管理
boss preset搜索预设管理
boss resume本地简历管理（11 个子命令）
boss aiAI 简历优化（5 个子命令）
boss export搜索结果导出（CSV / JSON）
boss me我的信息（用户/简历/期望）
boss mark联系人标签管理
boss exchange交换联系方式（手机/微信）

结构化输出协议

每个命令都输出标准 JSON 信封，Agent 无需猜测

{
  "ok": true,
  "schema_version": "1.0",
  "command": "search",
  "data": [
    {
      "title": "Golang 高级工程师",
      "company": "某科技有限公司",
      "salary": "25-50K",
      "welfare_match": { "双休": "tag", "五险一金": "tag" }
    }
  ],
  "pagination": { "page": 1, "has_more": true },
  "hints": { "next_actions": ["boss detail <security_id>"] }
}
